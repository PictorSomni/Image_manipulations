Procédure pour configurer l'accès SSH au Synology (pour Charles) :

1. Activer SSH sur le Synology :
   - Connecte-toi à l'interface DSM (ton navigateur).
   - Va dans "Panneau de configuration" > "Terminal et SNMP".
   - Coche "Activer le service SSH".
   - Clique sur "Appliquer".

2. Préparer l'accès pour moi (Gemini) :
   - Il me faut : l'adresse IP (ou nom de domaine) du Synology, ton nom d'utilisateur (avec droits d'accès SSH), et le port SSH (généralement 22, mais il vaut mieux le changer pour la sécurité).
   - NE ME DONNE PAS ton mot de passe ici !

3. Enregistrer les identifiants en toute sécurité :
   - Une fois que le service est activé, nous utiliserons `python Data/credentials.py set synology <ton_utilisateur>` dans ton terminal.
   - Cela ouvrira une boîte de dialogue sécurisée pour que tu saisisses ton mot de passe sans qu'il apparaisse dans le chat.

4. Test :
   - Une fois enregistré, je pourrai tenter une connexion via ssh_command.

Dis-moi quand le service SSH est activé sur ton Synology, et donne-moi l'adresse IP et l'utilisateur que tu veux utiliser.
