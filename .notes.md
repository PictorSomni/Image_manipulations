1. Opérations Git — le plus gros manque
Claude Code peut faire git status, git diff, git commit, git push, créer des PRs, lire le log… Rien de tel dans vos apps.

2. Édition partielle de fichiers (Edit)
Claude Code modifie des portions de fichier avec remplacement de chaîne précis. Vous n'avez que create_file qui réécrit le fichier entier — risqué sur des gros fichiers.

3. Recherche dans les contenus (Grep)
Recherche regex dans le contenu des fichiers (ex. trouver tous les fichiers contenant def main). Absent.

4. Recherche par motif (Glob)
Chercher des fichiers par pattern (**/*.py, src/**/*.json). Absent — list_folder_contents liste juste un dossier.

5. Suivi de tâches (TodoWrite)
Listes de tâches avec états (todo / in-progress / done) pour les travaux multi-étapes. Absent (votre mémoire ne remplace pas ça).

6. Lecture PDF
Claude Code peut lire des fichiers PDF. Aucun outil PDF dans vos apps.

7. Multi-agents / sous-agents
Claude Code peut déléguer du travail à des agents spécialisés en parallèle. Non applicable dans votre architecture actuelle.

8. Planification / mode Plan
Un mode de planification explicite avant implémentation. Absent.
