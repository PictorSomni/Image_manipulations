Procédure de rédaction pour fiches produits Prestashop (Mug personnalisé, etc.) :
1. Récapitulatif (max 800 caractères) - Onglet 1
2. Courte description (si nécessaire) - Onglet 1
3. Balise titre (max 70 caractères, doit finir par ' | MonObjet.be') - Onglet 2
4. Meta description (max 160 caractères) - Onglet 2
5. Mots-clefs pertinents.
